Glossaire

  • PFMP = Période de Formation en Milieu Proffessionel. Il s'agit de stage en entreprise qui sont souvent de longue duré 3-4 semaines voir 7 semaines pour les plus long.
  • VLAN = Virtual Local Area Netnork.
  • LAN = Local Aréa Netnork.
  • VM = Une Virtual Machine est héberbé au sein d'un odinateur.
  • Switch/Commutateur= Appareil permet d'interconecter des STA à un ou plusieurs réseaux entre eux le plus souvent avec des câbles RJ-45.
  • STA = Est une station de travail est une unité fonctionnelle informatique ayant habituellement des capacités de traitement spécialisé et intégrant des organes d'entrées-sorties orientés vers un utilisateur.
  • RJ-45 = Type de câble utilisé en majorité pour la conection Ethernet/Internet.
  • Windows 10 = Windows est un sytème d'exploitation (permet de rendre fonctionelle un ordinateur).
  • OS = Operating System, exemple : Windows 10.
  • Serveur = Ordinateur dédié à l'accomplissemant des tâches et services.
  • Serveur de Stockage = Un serveur de stockage vas garder vos données sur des disques dur pour vous.
  • Disque dur = Partie de l'ordinateur qui stocke les données.
  • BAC SN option RISC= Système Numérique option Réseaux Infrasctucture et Système Communicant.
  • CFA = Centre de Formation Académique.
  • BIA = Brevet d' Initiation Aéronotique.
  • GTC = Gestion Technique Centralisée.
  • SSH = Secure SHell est à la fois un programme informatique et un protocole de communication sécurisé.
  • Câble console/series = Ce type de câble est souvent utilisé pour se connecter à une machine en tant que admnistrateur pour y aplliquer des mises à jours ou des nouvelles stratégies, il permet d'accéder directement aux fichier de configuration à condition d'avoir les droits d'administrateur de l'appareil.
  • Admnistrateur = Un admnistrateur est la personne qui a les plus hauts droits de modifications et d'applications sur un système, il règle les soucis et configure de nouvelles stratégies.
  • ESXI = ESXI est un hyperviseur de typé 1, l'admnistrateur poura créer directement les VM à distance, et pour l'utilisateur il poura se connecter à la machine via un navigateur ou l'applicaction dédiée.
  • Hyperviseurs = Les hyperviseurs de type 1 sont aussi appelés hyperviseurs natifs ou bare-metal, car ils s'exécutent directement sur le matériel physique. Ils sont généralement plus performants et plus sécurisés que les hyperviseurs de type 2, qui s'exécutent sur un système d'exploitation hôte.
